# Home Page Image Gallery

This folder contains images that will be automatically displayed in the home page gallery carousel.

## How to Add Images

1. Simply add image files to this folder
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`, `.svg`
3. Images will be displayed in alphabetical order by filename
4. Recommended image dimensions: 1920x800 pixels (or similar aspect ratio)

## Recommended Image Naming Convention

For better organization, consider naming your images with a numbering prefix:
- `01_hall_exterior.jpg`
- `02_room_interior.jpg`
- `03_common_area.jpg`
- `04_study_room.jpg`
- etc.

## Tips for Best Results

1. **Optimize images** - Compress images for web to ensure fast loading
2. **Use consistent aspect ratios** - All images look best when they have similar proportions
3. **High quality images** - Use high-resolution images (minimum 1280px width)
4. **Descriptive filenames** - Use meaningful names for easy management

## Gallery Features

- **Auto-slideshow**: Images cycle automatically every 5 seconds
- **Manual navigation**: Users can click arrows or thumbnails to navigate
- **Responsive**: Gallery adapts to all screen sizes
- **Zoom effect**: Subtle zoom animation on active slide

## No Images?

If this folder is empty or contains no valid image files, the gallery section will not be displayed on the home page.
